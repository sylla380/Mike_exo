TP1. Ex10. Chess

Temps de référence15 mn
Temps estimé25 mn
Temps max35 mn

Enoncé

Soit un échiquier classique :

Construire un programme permettant de savoir, en fonctions de coordonnées saisies si :

  • une case est blanche (afficher B) ou noire (afficher N)
  • le déplacement d'un cavalier est valide ou pas (afficher V pour valide ou I pour invalide)

Ne pas utiliser de "if", mais uniquement des expressions logiques et l'opérateur ternaire ?:. L'utilisateur devra indiquer sur la ligne de commande quelle option il souhaite exécuter (1 pour déterminer la couleur de la case - blanche ou noire, 2 pour déterminer la validité d'un déplacement – correct ou incorrect) et les coordonnées attendues (ligne puis colonne).

Exemple de résulat(s) attendu(s)

ENTRER DANS L'INPUT : Mike 1 1 2

	Result : N

ENTRER DANS L'INPUT : Mike 2 4 5 2 6

	Result : V

ENTRER DANS L'INPUT : Mike 2 4 5 1 1

	Result : I

Travail attendu

  1. Dans le répertoire 'tp1', créer un sous répertoire 'rent'
  2. Créer un fichier rent.c
  3. Implémenter le programme dans le fichier c créé
  4. Le compiler, corriger les erreurs si besoin.
  5. Le copier dans son répertoire de rendu.
