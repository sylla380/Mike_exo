TP1. Ex9. Ip address

Temps de référence10 mn
Temps estimé20 mn
Temps max30 mn

Enoncé

Une adresse IP est un nombre de 32 bits identifiant une machine connectée a un réseau. Ce nombre est décomposable en 4 valeurs décimales non signé(correspondant a 4 octets). Par exemple, le nombre FFFF0000 (en hexadécimal) correspond a l'adresse : FF.FF.00.00. Construire un programme, utilisant obligatoirement des opérateurs binaires permettant de simuler une telle décomposition a partir d'un entier long fourni en décimal. Vous utiliserez la fonction strtoul() pour convertir le décimal fourni en ligne de commande.

ENTRER DANS L'INPUT : Mike 4294901760

	Result : 255.255.0.0

ENTRER DANS L'INPUT : Mike 2130706433

	Result : 127.0.0.1

Travail attendu

  1. Dans le répertoire 'tp1', créer un sous répertoire 'rent'
  2. Créer un fichier rent.c
  3. Implémenter le programme dans le fichier c créé
  4. Le compiler, corriger les erreurs si besoin.
  5. Le copier dans son répertoire de rendu.
