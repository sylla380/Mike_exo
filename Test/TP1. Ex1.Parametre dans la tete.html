TP1. Ex1. Recuperation des valeurs

Temps de référence3 mn
Temps estimé10 mn
Temps max15 mn

Enoncé

En PHP, comme dans la plupart des langages, il est possible de transmettre des données au programme. Pour cela il suffit juste de les ajouter dans un formulaire.

Ces données sont transmises dans les variables (Super GLOBAL) gerer par le serveur. La variable $_GET ou $_POST, de type array, contient les valeurs de ces données.

Le programme ci-dessous afficher les differentes données celon le language.

Si le programme fonctionne, lorsque vous le lancer avec moins de deux paramètres le message suivant apparaît : Nombre de paramètres insuffisants. Si "Mike" n'ai pas envoyer le message suivant apparaît : Premier valeur incorrect, sinon si les deux premiers paramètres sont affichés, préfixé de « Paramètre 1 : » et « Paramètre 2 : ». Si plus de trois paramètres le message suivant apparaît : Nombre de paramètres trop important.

ENTRER DANS L'INPUT : Mike 1 22

		Parametre 1 : 1
		Parametre 2 : 22

ENTRER DANS L'INPUT : Mike 22 banane

		Parametre 1 : 22
		Parametre 2 : banane
